# 豆包风格输入框设计计划

## 参考分析

### 豆包输入框特点
1. **布局结构**
   - 左侧垂直排列工具按钮（文件上传等）
   - 右侧主要输入区域
   - 输入框内嵌入发送按钮

2. **视觉设计**
   - 工具按钮：圆角矩形，简洁设计
   - 输入框：圆角设计，有边框
   - 发送按钮：小型圆形/圆角按钮
   - 整体风格：简洁、现代、扁平化

3. **交互效果**
   - 输入框聚焦时边框高亮
   - 按钮悬停效果
   - 输入框自适应高度

## 实现方案

### 1. 布局结构
- 左侧工具栏：垂直排列文件上传按钮
- 右侧输入区域：包含文本输入框和嵌入的发送按钮

### 2. 样式设计
- **工具按钮**：36px×36px，圆角8px，浅灰色背景
- **输入框**：圆角12px，边框1px，自适应高度
- **发送按钮**：32px×32px，圆角8px，浅蓝色背景

### 3. 颜色适配
- 背景色：保持项目的深色主题（#1e293b）
- 文字色：白色（#f8fafc）
- 边框色：浅灰色（rgba(255,255,255,0.1)）
- 高亮色：项目主色（#00c8ff）

### 4. 交互效果
- 输入框聚焦：边框变为主色，添加轻微发光效果
- 按钮悬停：背景色变亮
- 输入框自适应：根据内容高度自动调整
