# 修复计划：API调用、按钮样式和输入框对齐

## 问题分析

### 1. 图片生成API调用失败
- 当前使用的API可能不稳定或已失效
- 需要更换更可靠的免费图片生成API

### 2. 按钮样式需要扁平化设计
- 上传按钮和提交按钮需要更简洁的扁平化风格
- 移除渐变和阴影效果

### 3. 输入框对齐问题
- 上传按钮、输入框、提交按钮需要垂直对齐
- 确保在不同屏幕尺寸下都能正确对齐

## 修复方案

### 1. 修复API调用
- 更换为另一个免费图片生成API（如Hugging Face或其他可靠服务）
- 添加更好的错误处理和日志记录

### 2. 优化按钮样式
- 上传按钮：扁平化设计，悬停效果
- 提交按钮：扁平化设计，发送图标

### 3. 优化输入框对齐
- 确保所有元素在同一行垂直居中
- 添加响应式设计支持

## 修改文件

1. **js/app-detail.js** - 修复API调用
2. **app-detail.html** - 优化按钮样式和输入框布局
